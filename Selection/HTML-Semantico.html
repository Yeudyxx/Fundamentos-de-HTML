HTML Semántico

HTML semántico se refiere al uso de etiquetas que describen el significado del contenido que encierran. En lugar de usar genéricamente <div>, se utilizan etiquetas como:

  • <header> – Encabezado de una página o sección
  • <nav> – Menú de navegación
  • <section> – Sección temática del documento
  • <article> – Contenido independiente o autónomo
  • <aside> – Contenido complementario
  • <footer> – Pie de página

Estas etiquetas mejoran la accesibilidad y el SEO de una web.
