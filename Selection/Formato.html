Formato en HTML

HTML permite aplicar diferentes tipos de formato al texto utilizando etiquetas como:

  • <b> o <strong> para texto en negrita.
  • <i> o <em> para texto en cursiva.
  • <u> para subrayado (aunque no se recomienda mucho hoy en día).
  • <mark> para resaltar texto.

También puedes utilizar encabezados jerárquicos desde <h1> hasta <h6> para títulos y subtítulos.
