Curso de Introducción a HTML

HTML (HyperText Markup Language) es el lenguaje fundamental para crear páginas web. Es un lenguaje de marcado que estructura el contenido mediante etiquetas.

  • Listas ordenadas y desordenadas

Estructura básica de HTML

Todos los documentos HTML siguen una estructura fundamental:

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Título de la página</title>
</head>
<body>
    <!-- Contenido visible de la página -->
</body>
</html>

HTML Semántico

El HTML semántico utiliza etiquetas que describen su significado tanto para el navegador como para el desarrollador:

  • <header> - Cabecera de la página o sección
  • <nav> - Navegación principal
  • <main> - Contenido principal
  • <section> - Sección temática
  • <article> - Contenido independiente
  • <aside> - Contenido relacionado indirectamente
  • <footer> - Pie de página o sección

¿Por qué usar HTML semántico? Mejora la accesibilidad, el SEO y la mantenibilidad del código.
