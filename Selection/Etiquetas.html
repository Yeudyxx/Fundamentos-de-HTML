Etiquetas en HTML

Una etiqueta HTML es un componente del lenguaje que define el significado y la estructura del contenido. Las etiquetas van entre corchetes angulares: <nombre>.

Ejemplo de etiqueta de párrafo:

<p>Este es un párrafo</p>

Muchas etiquetas tienen una etiqueta de apertura y una de cierre, aunque algunas son auto-cerradas (como <img> o <br>).
