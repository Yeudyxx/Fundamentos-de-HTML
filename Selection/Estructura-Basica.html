Estructura Básica de un Documento HTML

Todo archivo HTML debe seguir una estructura básica, que incluye las siguientes partes:


<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <title>Título del Documento</title>
  </head>
  <body>
    <h1>Contenido principal</h1>
  </body>
</html>

Esta estructura garantiza que el navegador interprete correctamente el contenido.
